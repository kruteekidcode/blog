Python เป็นภาษาโปรแกรมที่ได้รับความนิยมสูงสุดในโลก เหมาะสำหรับผู้เริ่มต้นเพราะมี syntax ที่อ่านง่ายเหมือนภาษาอังกฤษ

## ทำไมต้อง Python?

- **เรียนรู้ง่าย** — syntax สะอาด อ่านง่าย ไม่ซับซ้อน
- **ใช้งานได้หลากหลาย** — เว็บ, AI, Data Science, Automation
- **ชุมชนใหญ่** — มี library และ tutorial มากมาย
- **ตลาดแรงงานต้องการ** — เป็นภาษาที่นายจ้างต้องการมากที่สุด

## ติดตั้ง Python

เข้าไปที่ [python.org](https://python.org) แล้วดาวน์โหลดเวอร์ชันล่าสุด ติดตั้งตามขั้นตอนปกติ

ทดสอบว่าติดตั้งสำเร็จ:

```bash
python --version
```

## เขียนโปรแกรมแรก

สร้างไฟล์ `hello.py` แล้วพิมพ์:

```python
# โปรแกรมแรกของเรา
print("สวัสดี Python!")

# ตัวแปรและชนิดข้อมูล
name = "ครูทีกิด"
age = 30
is_teacher = True

print(f"ชื่อ: {name}, อายุ: {age}")
```

## ตัวแปรและชนิดข้อมูลพื้นฐาน

Python มีชนิดข้อมูลหลักๆ ดังนี้:

| ชนิดข้อมูล | ตัวอย่าง | อธิบาย |
|-----------|---------|--------|
| `str` | `"hello"` | ข้อความ |
| `int` | `42` | จำนวนเต็ม |
| `float` | `3.14` | ทศนิยม |
| `bool` | `True` | ค่าจริง/เท็จ |
| `list` | `[1, 2, 3]` | รายการ |

## เงื่อนไขและลูป

```python
# เงื่อนไข if-else
score = 85

if score >= 80:
    print("เกรด A")
elif score >= 70:
    print("เกรด B")
else:
    print("ต้องพยายามเพิ่ม")

# ลูป for
fruits = ["แอปเปิ้ล", "กล้วย", "ส้ม"]
for fruit in fruits:
    print(f"ผลไม้: {fruit}")
```

## สรุป

Python เป็นภาษาที่เหมาะมากสำหรับผู้เริ่มต้น เริ่มจากพื้นฐานเหล่านี้ แล้วค่อยๆ ต่อยอดไปยังหัวข้อที่สนใจ ไม่ว่าจะเป็น Web Development, Data Science หรือ AI
