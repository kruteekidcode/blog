## ภาพรวมหลักสูตร

หลักสูตรนี้ออกแบบมาสำหรับผู้ที่ไม่เคยเขียนโปรแกรมมาก่อนเลย เราจะเริ่มต้นตั้งแต่พื้นฐานที่สุด และค่อยๆ สร้างทักษะไปจนสามารถเขียนโปรแกรม Python ที่มีประโยชน์ได้จริง

## สิ่งที่จะได้เรียนรู้

- ติดตั้งและตั้งค่า Python บนเครื่องคอมพิวเตอร์
- ตัวแปร ชนิดข้อมูล และ operators
- เงื่อนไข (if-else) และลูป (for, while)
- ฟังก์ชัน (functions) และ modules
- การอ่านเขียนไฟล์
- พื้นฐาน Object-Oriented Programming
- โปรเจกต์จริง 3 โปรเจกต์

## เหมาะสำหรับใคร

- ผู้ที่ไม่เคยเขียนโปรแกรมมาก่อน
- นักเรียน นักศึกษาที่อยากเริ่มเรียน coding
- ผู้ที่ต้องการเปลี่ยนสายงานมาด้าน IT
- ครูผู้สอนที่ต้องการสอน Python

## ระยะเวลา

ประมาณ 20 ชั่วโมง (เรียนตามสะดวก)
