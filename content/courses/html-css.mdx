## ภาพรวมหลักสูตร

เรียนรู้การสร้างเว็บไซต์ตั้งแต่พื้นฐาน ด้วย HTML5 และ CSS3 หลักสูตรนี้จะพาคุณสร้างเว็บไซต์ที่สวยงาม ใช้งานได้จริง และรองรับทุกขนาดหน้าจอ

## สิ่งที่จะได้เรียนรู้

- พื้นฐาน HTML5 — โครงสร้าง, tags, semantic HTML
- พื้นฐาน CSS3 — selectors, properties, box model
- Flexbox และ Grid Layout
- Responsive Design สำหรับมือถือและ tablet
- CSS Animation และ Transition
- ปรับแต่งฟอนต์และสี
- โปรเจกต์จริง: สร้างเว็บไซต์ portfolio

## เหมาะสำหรับใคร

- ผู้ที่ต้องการสร้างเว็บไซต์เป็นครั้งแรก
- ผู้ที่อยากเข้าใจพื้นฐานก่อนเรียน JavaScript
- ครูผู้สอนวิชาคอมพิวเตอร์
- ผู้ที่ต้องการทำเว็บไซต์ส่วนตัวหรือธุรกิจ
